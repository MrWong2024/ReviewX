# ReviewX Backend

ReviewX backend is currently a public NestJS skeleton with configuration loading,
basic error handling, and a minimal health check. No business modules are
implemented yet.

## Common Commands

- `npm run start:dev`
- `npm run build`
- `npm run lint`
- `npm test`
- `npm run test:e2e`

## Notes

- The current scope is infrastructure only.
- Business modules and database integration are intentionally not initialized in
  this task.
